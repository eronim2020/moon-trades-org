# Petrognosis MVP - Mineral Analysis Platform
Moon Trades SRL | AI-Powered Spectral Analysis

## Quick Start

```bash
pip install -r requirements.txt
streamlit run petrognosis_mvp.py
```

## What This Demo Shows

- **Live Demo Tab**: Real-time mineral classification from spectral data
- **Spectral Library Tab**: NIR reflectance signatures (610-940nm) for 5 minerals
- **About Tab**: Platform overview + Geostud use case

## Technical Stack

- Raspberry Pi 5 + Hailo AI HAT+ + AS7265x NIR Sensor
- CNN trained on USGS Spectral Library
- Cost: €420-760 per unit
